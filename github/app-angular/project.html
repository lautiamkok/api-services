Repo App
  • List Repos
  • Create Repo
  • Pull Request
  • Count Emoji

This application uses official GitHub API v3 at https://developer.github.com/v3/ for following tasks:

  • 1. list all the user repositories.
  • 2. allow the user to input a GitHub repository
  • 3. fetch all pull requests from a repo.
  • 4. count emoji in a pull request review comment.
  • 5. show how many times each emoji appears. ref: https://developer.github.com/v3/emojis/
